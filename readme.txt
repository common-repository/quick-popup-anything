A lightweight, responsive and easy to use popup plugin. Insert shortcodes & other content from your favorite plugins to create Quick Popup.

== Description ==

Quickly add powerful modal popups for your WordPress blog or website. Powerful yet, easy to use this plugin will help you to add any shortcode, contact from or grab your visitors attention to introduce them your offers, discounts or other kind of promotional notices.

* Responsive popup plugin
* Choose between several button position
* HTML popup
* Image popup
* Customize the look and feel of the popup
* Work well with contact form plugin
* Display shortcode output in the popup
* Responsive popup

== Installation ==

1. Upload the 'mi-quick-popup-anything' folder to the '/wp-content/plugins/' directory.
2. Activate the "Quick Popup Anything" list plugin through the 'Plugins' menu in WordPress.
3. Navigate to Dashboard Menu->Quick Popup to configure the plugin.
